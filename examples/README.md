# 範例專案：台南安順場址污染示範（模擬）

`anshun_demo_project.json` — 一個可直接匯入的 Web GIS 專案，依**台南中石化（原台鹼）安順整治場址**的污染特性，示範本平台能產出的各式圖表。

> ⚠️ **所有濃度為模擬值**，僅供功能展示。污染物種類、台灣土壤／地下水污染管制／監測標準、場址地理位置依公開資料；採樣點位置為示意配置，非實際座標。

## 如何使用

1. 開啟 Web GIS（<https://gis.tinghaochang.com> 或本機），登入後在專案列點 **匯入專案**。
2. 選擇 `anshun_demo_project.json`。
3. 底圖預設為 Google 衛星混合圖，視野已框在安南區廠區＋海水貯水池。

## 場址背景

中石化安順廠以汞電解法製鹼氯、並生產五氯酚鈉（副產物含戴奧辛）。**汞、戴奧辛、五氯酚**滲入土壤與地下水；**海水貯水池**底泥為最嚴重熱點。原污染面積約 37 公頃，土壤汞、戴奧辛最高分別達 9,950 mg/kg、64,100,000 ng I-TEQ/kg。

## 圖層與展示重點

| 圖層 | 類型 | 展示功能 |
|------|------|----------|
| 場址範圍與海水貯水池 | 多邊形 | 場址邊界、貯水池 A/B 區範圍標示 |
| 地下水 汞／五氯酚 等濃度線（2013–2024） | 等濃度線 | **多污染物 × 多期**合併圖層：可切換 汞／五氯酚、用 ◀▶ 切換 2013→2018→2024，看污染團隨整治**逐期縮小**；含管制（0.02 / 0.08）與監測（0.01 / 0.04）標準閾值線、分級配色圖例 |
| 地下水監測井 | 點 | 12 口井，屬性表內含三期 汞／五氯酚 濃度；可改 IDW／TIN／Kriging／Indicator 內插模型重算 |
| 土壤 汞／戴奧辛 超標點位圖（預設關閉） | 點 | **點位超標圖**：依管制/監測標準分級紅(超管制)/橙(超監測)/綠(合格)，可切換 汞／戴奧辛、依採樣批次（貯水池底泥／廠區表土）不同點形狀與開關 |
| 土壤採樣點位（預設關閉） | 點 | 超標圖的來源資料（14 點，2 批次） |
| 土壤污染調查（汞・垂向，預設關閉） | 點 | **垂向多深度鑽探**：屬性表內每點 0–4m 各層汞濃度。點屬性表「3D 體積」可開 3D：堆疊切片／平滑曲面、逐層超管制(≥20)面積與**體積（≈ 417,000 m³）**、含障礙物（舊廠房基礎）排除 |

## 推薦展示動線

1. 預設畫面：場址範圍 + 地下水汞等濃度線（2013，污染團最大）。
2. 展開「地下水…等濃度線」圖層 → ◀▶ 切到 2024，看汞污染團大幅縮小（整治成效）；切換到五氯酚看另一污染物。
3. 開啟「土壤 汞／戴奧辛 超標點位圖」→ 切換 汞／戴奧辛，看貯水池底泥點全紅（超管制）。
4. 開啟「土壤污染調查」圖層 → 屬性表 → 3D 體積，看汞污染垂向集中於上層、體積估算。

## 資料來源

- 污染背景：中石化安順廠污染案（維基百科）、環境部土壤及地下水污染整治網。
- 標準：環境部「土壤污染管制／監測標準」「地下水污染管制／監測標準」（第二類）——
  汞 土壤 20/10、地下水 0.02/0.01；五氯酚 土壤 200/200、地下水 0.08/0.04；戴奧辛 土壤 1000/1000 ng-TEQ/kg。

## 重建來源資料

`anshun_seed_source_layers.py` 為**來源四圖層**（場址、地下水井、土壤採樣、垂向鑽探）的產生器（以高斯熱羽模擬濃度場）。等濃度線與超標圖兩個衍生圖層是在 App 內按「生成」鈕產出後，連同來源一起匯出成上述 JSON。
