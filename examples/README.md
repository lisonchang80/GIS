# 範例專案：台南安順場址污染示範（模擬）

`anshun_demo_project.json` — 一個可直接匯入的 Web GIS 專案，依**台南中石化（原台鹼）安順整治場址**的污染特性，示範本平台能產出的各式圖表。

> ⚠️ **所有濃度為模擬值**，僅供功能展示。污染物種類、台灣土壤／地下水污染管制／監測標準、已公開的最高值（汞 9,950 mg/kg、戴奧辛 64,100,000 ng I-TEQ/kg）依公開資料；場址定位於真實的安南區顯宮段（~120.135°E, 23.048°N），**場址範圍採真實地號**（顯宮段 270–300）；監測點位為示意配置。

## 如何使用

1. 開啟 Web GIS（<https://gis.tinghaochang.com> 或本機），登入後在專案列點 **匯入專案**，選 `anshun_demo_project.json`。
2. 底圖為 Google 衛星混合圖，視野已框在顯宮段場址。預設顯示：地號場址範圍＋海水貯水池＋監測井＋地下水位等水位線/流向。

## 場址背景

中石化安順廠以汞電解法製鹼氯、並生產五氯酚鈉（副產物含戴奧辛）。**汞、戴奧辛、五氯酚**滲入土壤與地下水；**海水貯水池**底泥為最嚴重熱點。原公告土壤污染面積約 37 公頃。

## 圖層與展示重點（8 層）

| 圖層 | 類型 | 展示功能 |
|------|------|----------|
| 整治場址範圍（顯宮段地號） | 多邊形 | **「輸入地號新增多邊形」**：31 筆真實地籍地號（顯宮段 270–300，經地籍 API 取得）構成場址街廓 |
| 海水貯水池（底泥熱點） | 多邊形 | 貯水池 A／B 區範圍 |
| 地下水位 等水位線／流向（2024-11） | 線 | **水文監測 → 等水位線**：水位高程等值線（z 高程＝地表高程−水位埋深）＋**流向箭頭**，水位由內陸（東南）向海（西北）遞降 |
| 地下水 汞／五氯酚 等濃度線（2013–2024，預設關閉） | 等濃度線 | **多污染物 × 多期**合併等濃度線：切換 汞／五氯酚、◀▶ 切 2013→18→24 看污染團隨整治**逐期縮小**；含管制/監測標準閾值線、分級圖例 |
| 地下水監測井 | 點 | 12 口井，屬性表含水位（水文監測分頁）＋三期 汞／五氯酚 濃度（地下水濃度監測分頁）；可換 IDW/TIN/Kriging/Indicator |
| 土壤 汞／戴奧辛 超標點位圖（預設關閉） | 點 | **點位超標圖**：紅(超管制)/橙(超監測)/綠(合格)，切換 汞／戴奧辛、依批次（底泥圓點／表土方點）不同點形 |
| 土壤採樣點位（預設關閉） | 點 | 超標圖來源（14 點，2 批次） |
| 土壤污染調查（汞・垂向，預設關閉） | 點 | **垂向多深度鑽探**，點屬性表「3D 體積」開 3D：堆疊切片／平滑曲面、逐層超管制(≥20)面積與**體積（≈ 417,000 m³）**、含障礙物排除 |

## 推薦展示動線

1. 預設畫面：地號場址範圍＋海水貯水池＋等水位線/流向（看地下水由東南流向西北的海）。
2. 開「地下水…等濃度線」→ 預設 2013 汞污染團最大 → ◀▶ 切 2024 看縮小 → 切五氯酚。
3. 開「土壤…超標點位圖」→ 切 汞／戴奧辛，看貯水池底泥全紅（超管制）。
4. 開「土壤污染調查」→ 屬性表「3D 體積」→ 看汞污染垂向集中於上層 + 體積估算。

## 資料／標準來源

- 污染背景：中石化安順廠污染案（維基百科）、環境部土壤及地下水污染整治網、臺南市環保局。
- 標準（環境部）：汞 土壤20/10・地下水0.02/0.01；五氯酚 土壤200/200・地下水0.08/0.04；戴奧辛 土壤1000/1000 ng-TEQ/kg。
- 地號／地籍：twland.ronny.tw（與 App「輸入地號新增多邊形」同來源）。

## 重建來源資料

`anshun_seed_source_layers.py` 為**來源圖層**產生器：抓真實顯宮段地號當場址範圍、以高斯熱羽模擬各污染場、置於真實場址座標、並含地下水位（流向）。等濃度線／等水位線／超標圖三個衍生圖層是在 App 內按「生成」鈕產出後，連同來源匯出成此 JSON。

> 註：垂向鑽探的汞峰值取中等值（90 mg/kg）以使等濃度線/體積計算穩定；表層抓樣層仍保留已公開最高值（9,950）。前者是因 3D 逐層面積在污染濃度整片超過閾值時，現行 turf.isobands 計算會退化（已另開修復任務），與「系統鑽探 vs 定點抓樣」的真實差異一致。
