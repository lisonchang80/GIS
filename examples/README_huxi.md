# 範例專案：澎湖中油湖西油庫漏油場址示範（模擬）

`huxi_demo_project.json` — 一個可直接匯入的 Web GIS 專案，依**澎湖縣湖西供油中心（中油湖西油庫）漏油整治場址**的污染特性，示範本平台在「石油類污染（TPH／苯／MTBE）」場景能產出的各式圖表。為 [台南安順場址示範](README.md) 的姊妹範例（汞／戴奧辛 vs 石油類）。

> 來源：澎湖縣政府「中油-湖西油庫漏油事件訊息專區」、環境部土壤及地下水污染整治網／場址資訊雲端展示查詢網。
> 場址位於**澎湖縣湖西鄉湖西村102號之2**，西北側約 1.7 km 為海岸。

> ⚠️ **所有濃度為模擬值**，僅供功能展示。污染物種類、台灣土壤／地下水污染管制標準、漏油量（68.4 公秉）與 2018 年公告地下水最高濃度（TPH 33.2、苯 4.47、萘 0.515、**MTBE 120** mg/L）依公開資料。
> **場址範圍採油庫實際 footprint**（OpenStreetMap way 508438551「中油湖西油庫」，landuse=industrial，~6.7 公頃）——油庫為單一工業地塊，未對應「湖西段」農地地號，且整治場址逐筆公告地號未公開列出，故以實際範圍定界（與安順採真實地籍**地號**逐筆繪製的方式不同）；監測點位為示意配置。

## 如何使用

1. 開啟 Web GIS（<https://gis.tinghaochang.com> 或本機），登入後在專案列點 **匯入專案**，選 `huxi_demo_project.json`。
2. 底圖為 Google 衛星混合圖，視野已框在油庫。預設顯示：場址範圍＋漏油污染源（儲槽區）＋地下水監測井＋地下水位等水位線/流向。

## 場址背景

2017 年 6–7 月中油湖西油庫漏油，累計約 **68.4 公秉**，污染土壤與地下水。**2018-12-10 公告為整治場址**，整治期限至 **2024-02-19**。漏油以**汽油類**為主，地下水驗出 **TPH、苯（BTEX）、MTBE（汽油添加劑，最易隨地下水擴散）**。

## 污染物與標準

| 污染物 | 介質 | 管制標準 | 監測/篩選 | 備註 |
|--------|------|----------|-----------|------|
| 總石油碳氫化合物 TPH | 土壤 | 1,000 mg/kg | 500 | 油類整體指標 |
| TPH | 地下水 | 10 mg/L | 5 | |
| 苯 (benzene) | 土壤 | 5 mg/kg | 2.5 | 致癌性 BTEX |
| 苯 | 地下水 | 0.05 mg/L | 0.005 | |
| MTBE | 地下水 | 1 mg/L | 0.5 | 高移動性、羽流最大 |

（監測/篩選值取管制值之半，為展示分級用。）

## 圖層與展示重點（8 層）

| 圖層 | 類型 | 展示功能 |
|------|------|----------|
| 地下水監測井 | 點 | 12 口井，屬性表含水位（水文監測分頁）＋三期 TPH／苯／MTBE 濃度（地下水濃度監測分頁）；可換 IDW/TIN/Kriging/Indicator |
| 土壤 TPH／苯 超標點位圖（預設關閉） | 點 | **點位超標圖**：紅(超管制)/橙(超監測)/綠(合格)，切換 TPH／苯、依批次（儲槽區圓點／周邊方點）不同點形 |
| 土壤採樣點位（預設關閉） | 點 | 超標圖來源（12 點，2 批次） |
| 土壤污染調查（TPH・垂向，預設關閉） | 點 | **垂向多深度鑽探**，點屬性表「3D 體積」開 3D：堆疊切片／平滑曲面、逐層超管制(≥1000)面積與**體積（≈ 146,000 m³）**、含障礙物排除（儲槽混凝土基座）與缺層補估 |
| 地下水位 等水位線／流向（2024-02） | 線 | **水文監測 → 等水位線**：水位高程等值線（z 高程＝地表高程−水位埋深，約 1–2 m）＋**流向箭頭**，水位由內陸（東南）向海（西北）遞降 |
| 地下水 TPH／MTBE 等濃度線（2018–2024，預設關閉） | 等濃度線 | **多污染物 × 多期**合併等濃度線：切換 TPH／MTBE、◀▶ 切 2018→21→24 看污染團隨整治**逐期縮小**；含管制/監測標準閾值線、分級圖例 |
| 漏油污染源（儲槽區） | 多邊形 | 漏油點位示意 |
| 整治場址範圍（中油湖西油庫） | 多邊形 | 油庫實際 footprint（OSM，~6.7 公頃） |

## 推薦展示動線

1. 預設畫面：場址範圍＋漏油污染源＋等水位線/流向（看地下水由東南流向西北的海）。
2. 開「地下水…等濃度線」→ 預設 2018 TPH 污染團最大 → ◀▶ 切 2024 看縮小、切 MTBE 看高移動性羽流往下游（西北）延伸。
3. 開「土壤…超標點位圖」→ 切 TPH／苯，看儲槽區源頭全紅（超管制）。
4. 開「土壤污染調查」→ 屬性表「3D 體積」→ 看 TPH 垂向集中於上層（油類滲流）+ 體積估算。

## 資料/標準來源

- 污染背景：澎湖縣政府中油-湖西油庫漏油事件專區、環境部土壤及地下水污染整治網。
- 標準（環境部）：TPH 土壤1000・地下水10；苯 土壤5・地下水0.05；MTBE 地下水1（mg/L 或 mg/kg）。
- 場址 footprint：OpenStreetMap way 508438551（中油湖西油庫，landuse=industrial）。

## 重建來源資料與設計說明

`huxi_demo_project.py` 為**來源圖層**產生器：以油庫實際 footprint 當場址、漏油儲槽為污染源、以高斯熱羽模擬 TPH／苯／MTBE 場、置於真實場址座標（119.6544E,23.5762N）、並含地下水位（向西北海岸流向）。等濃度線／等水位線／超標圖三個衍生圖層在 App 內按「生成」鈕產出後，連同來源匯出成此 JSON。

> 設計註：
> 1. **MTBE 內插等濃度線採適中代表峰值**（與 TPH 同數量級比值），以免等濃度線過密、檔案肥大；2018 公告實測最高達 **120 mg/L**（極高移動性），此真實值於本說明揭露。
> 2. **苯不納入等濃度線圖**（苯峰值/監測標準比過大會使內插等值線爆量）；苯結果改於**監測井屬性表**與**土壤超標點位圖**呈現。
> 3. 垂向鑽探 TPH 峰值取中等（4,500 mg/kg）以使逐層面積/體積計算穩定（現行 turf.isobands 在整片超標時會退化，已另開修復任務）；表層抓樣（超標圖，純點不內插）保留石油類常見高值（TPH 數萬 mg/kg）。
